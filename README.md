# Rikol

Mobile-first spaced repetition app inspired by Anki.

## Commands

```bash
npm run dev
npm test
npm run build
npx playwright test
```

## Features

- Local-first IndexedDB storage.
- Review queue with Again, Hard, Good, Easy grading.
- Deck, card, tag, template, and progress management.
- JSON backup, CSV exchange, and best-effort Anki APKG import/export.
- PWA manifest and service worker.
